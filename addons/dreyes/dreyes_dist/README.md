# `dreyes_dist`

Modulo de formulario extendido para distribuidores.

## Incluye

- configuracion `basic` o `extended` por sitio web;
- captura obligatoria de datos fiscales y de contacto;
- ruta `/profile/complete`;
- redireccion posterior a login o signup cuando el registro extendido esta activo.

## Dependencias

- `dreyes_portal`
