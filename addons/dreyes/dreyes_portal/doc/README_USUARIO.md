# Que hace el modulo `dreyes_portal`

`dreyes_portal` controla la pantalla de acceso de clientes al sitio web de DReyes.

Permite:

- iniciar sesion;
- registrar nuevas cuentas;
- mostrar login y registro con imagen, colores y textos personalizados;
- enviar al usuario autenticado al home del sitio web.

Este modulo ya no administra formularios extendidos de captura de datos. Esa funcionalidad vive en `dreyes_dist`.
